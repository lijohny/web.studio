# **App Name**: Web Studio Ignite

## Core Features:

- Hero Section Animation: Animated hero section on the homepage to immediately grab attention with the logo and tagline.
- Dynamic Portfolio Filtering: Filterable portfolio to easily showcase design, development, and SEO projects.
- Interactive Contact Form: Contact form with animated input fields and a dynamic submit button.
- SEO Content Generation: Use AI to suggest relevant keywords and phrases that enhance SEO, acting as a tool to assist the content creation.
- Smooth Scroll Navigation: Implement smooth scrolling for a modern, seamless browsing experience.
- Mobile-First Responsiveness: Ensure the website is fully responsive and optimized for mobile devices.
- Glow Effect Call-to-Action: Call-to-action buttons with glow hover effects to encourage user interaction.

## Style Guidelines:

- Primary color: Vibrant red (#FF4136) for bold highlights, inspired by the logo.
- Background color: Near-black (#111111) to create a dark, modern theme.
- Accent color: Electric white (#F0F0F0) to ensure readability and contrast against the dark background.
- Body and headline font: 'Space Grotesk', a sans-serif with a techy feel suitable for both headlines and body text.
- Use custom designed icons with neon glow effect to represent services.
- Minimalist layout with full-width sections and ample spacing.
- Use parallax scrolling and fade-in transitions for visual interest.
